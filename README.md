# Resume-Page
This project is a personal Resume  designed to showcase my profile, skills, projects, and education in a clean and responsive format.

# Setup Steps
1. Clone the repository.
2. Navigate to the project folder.
3. Open index.html in your browser.

# Usage
1. Highlighting Skills and Projects.
2. View Personal details.
